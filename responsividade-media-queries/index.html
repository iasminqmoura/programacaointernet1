Layout Web Responsivo

Este projeto foi desenvolvido para aplicar os conceitos de viewport, media queries, breakpoints, CSS Flexbox e CSS Grid, criando uma página institucional organizada, legível e adaptável para desktop, tablet e mobile.

Sobre o projeto

A proposta desta atividade é construir uma página que se comporte corretamente em diferentes tamanhos de tela, mantendo uma estrutura limpa e boa experiência de visualização em qualquer dispositivo.

Informações

Viewport

A página utiliza a configuração correta de viewport para se ajustar adequadamente aos diferentes dispositivos.

CSS Grid

A estrutura principal e a organização dos cards utilizam Grid para distribuir o conteúdo de forma clara e eficiente.

Flexbox

O menu de navegação e o alinhamento interno dos cards foram desenvolvidos com Flexbox.

Responsividade

O layout se adapta para desktop, tablet e mobile com o uso de media queries e breakpoints bem definidos.

Legibilidade

O conteúdo foi organizado para manter leitura confortável e espaçamentos adequados em qualquer largura de tela.

Boas práticas

O projeto utiliza CSS externo, reset, box-sizing, classes semânticas e evita excesso de medidas fixas.
