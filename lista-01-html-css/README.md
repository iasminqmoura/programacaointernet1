# Projeto HTML e CSS

## Objetivo do Projeto
Este projeto tem como objetivo praticar os conceitos básicos de desenvolvimento web utilizando **HTML5** e **CSS3**.  
Durante os exercícios foram aplicadas estruturas semânticas, seletores CSS e conceitos do **Box Model**, além de boas práticas de organização do código.

---

## Estrutura do Projeto

O repositório contém os seguintes arquivos:

- **index.html** → estrutura da página utilizando HTML5  
- **style.css** → arquivo responsável pela estilização da página  
- **README.md** → descrição do projeto e dos exercícios desenvolvidos  

---

## Exercício 1 – Estrutura Básica em HTML

Foi criado um arquivo **index.html** contendo:

- Estrutura completa de um documento HTML5
- Definição do idioma da página (`lang="pt-BR"`)
- Definição de charset (`UTF-8`)
- Um título principal utilizando a tag `<h1>`
- Um parágrafo utilizando a tag `<p>`

Este exercício teve como objetivo compreender a estrutura básica de uma página HTML.

---

## Exercício 2 – HTML Semântico

O conteúdo da página foi reorganizado utilizando **tags semânticas**, melhorando a organização e a leitura do código.

Foram utilizadas as seguintes tags:

- `<header>` → cabeçalho da página  
- `<nav>` → menu de navegação com três links fictícios  
- `<main>` → conteúdo principal da página  
- `<section>` → organização do conteúdo em seções  
- `<footer>` → rodapé da página  

Esse exercício teve como objetivo demonstrar boas práticas na estruturação de páginas HTML.

---

## Exercício 3 – CSS Básico e Seletores

No arquivo **style.css** foram aplicadas estilizações básicas utilizando diferentes tipos de seletores:

- Alteração da **cor de fundo da página**
- Estilização do título `<h1>` utilizando **seletor de tag**
- Estilização de um parágrafo utilizando **seletor de classe**
- Estilização de um elemento específico utilizando **seletor de id**

Este exercício teve como objetivo praticar o uso de **seletores CSS**.

---

## Exercício 4 – Box Model

Foi criada uma `<div>` com classe **card** para demonstrar o funcionamento do **Box Model** no CSS.

Foram aplicadas as seguintes propriedades:

- `width`
- `padding`
- `border`
- `margin`
- `box-sizing: border-box`

Este exercício teve como objetivo compreender como os elementos são organizados visualmente na página.

---

## Exercício 5 – Organização e Boas Práticas

Foram aplicadas boas práticas de desenvolvimento:

- Utilização de **CSS externo** (`style.css`)
- Uso de **nomes de classes claros e semânticos**
- **Indentação e organização do código**
- Separação adequada entre **estrutura (HTML)** e **estilo (CSS)**

---

## Tecnologias Utilizadas

- HTML5
